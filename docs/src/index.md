# Minerva.jl

Documentation for [Minerva.jl](https://github.com/fe-htt/Minerva.jl), a library for the 
[ModelingToolkit](https://docs.sciml.ai/ModelingToolkit/stable/) acausal modeling system to simulate the fuel cycle of fusion devices.

## Installation
To install Minerva:
- open a new folder
- activate a new julia environemnt ```activate .```
- enter package mode ```]```
- add the package to the environemnt using the GitHub URL ```add  https://github.com/fe-htt/Minerva.jl.git```
